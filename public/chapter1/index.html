JS 入門環境安裝

  1. 安裝 VScode->安裝插件->延伸模組(需要注意下載人數，避免下錯)
  • material icon theme(小圖示)
  1. 前端主要使用
  • HTML
  • CSS
  • JAVAScript(JS)
  1. 進到主控台就能寫 JAVAScript
  • 簡單測試 alret 指令跳出警告

  1. 創建.html 檔案，進入檔案輸入!會產生 HTML 基本檔案

  2. 安裝插件->live server，用來直接模擬 html 執行結果

  3. <script src = "檔案名稱.js"> </script>

  4. 在檔案名稱.js 輸入 console.log(“123”)在主控台印出 123

  5. 安裝 node.js->在終端機輸入 node->可以仿照控制台狀態->直接輸入node 檔案名稱.js就可以跑出結果

  6. 執行 JS = 執行環境(Runtime)
